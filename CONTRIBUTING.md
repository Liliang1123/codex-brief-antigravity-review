# Contributing to codex-brief-antigravity-review

Thank you for your interest in contributing to this project! This skill aims to set the standard for decoupled orchestrator-executor patterns in AI-assisted software engineering.

## How to Contribute

1. **Fork the Repository**: Clone your fork to your local environment.
2. **Setup Codex Local Path**:
   Link the skill locally to test your modifications:
   ```bash
   ln -s "$(pwd)" ~/.codex/skills/codex-brief-antigravity-review
   ```
3. **Make Your Changes**:
   - Keep `SKILL.md` clean, procedural, and focused on gates and responsibilities.
   - Keep templates and detailed instructions in the `references/` directory.
   - Avoid hardcoding user-specific paths or domains.
4. **Validate**:
   - Ensure the Mermaid flowcharts in `SKILL.md` render correctly.
   - Verify YAML frontmatter syntax.
5. **Submit a Pull Request (PR)**:
   - Provide a clear explanation of what improvements your changes introduce.
   - Mention the verification results.

## Code of Conduct

Please maintain professional and respectful communication when interacting with this repository and its contributors.
